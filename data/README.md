# Данные

Эта директория содержит данные, используемые в эксперименте.

## Структура

```
data/
├── telegram/          # Посты из Telegram каналов
│   └── posts.json     # Собранные посты
├── moex/              # Котировки MOEX
│   └── *.csv          # Файлы с котировками по тикерам
├── llm_cache/         # Кэш ответов LLM (для воспроизводимости)
└── results/           # Результаты экспериментов
    └── evaluation_results.json
```

## Воспроизводимость

Для воспроизведения результатов:

1. **Telegram данные**: 
   - Запустите сбор постов через `src/ingest_telegram.py` или используйте сохраненные данные
   - Формат: JSON с массивом постов

2. **MOEX данные**:
   - Загружаются автоматически через API при запуске эксперимента
   - Можно сохранить локально для ускорения повторных запусков

3. **LLM кэш**:
   - Кэшируется автоматически в `llm_cache/`
   - Позволяет воспроизводить результаты без повторных запросов к API

## Образцы данных

Если у вас есть образцы данных, разместите их здесь с соответствующими именами файлов.

## Примечания

- Данные из Telegram могут быть большими по размеру
- Кэш LLM помогает экономить на API запросах
- Котировки MOEX обновляются в реальном времени через API
