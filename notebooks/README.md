# Notebooks

Эта директория содержит Jupyter ноутбуки для исследования данных и проведения абляционных экспериментов.

## Использование

Создайте ноутбуки для:
- Исследования собранных данных
- Визуализации результатов
- Абляционных экспериментов (изменение параметров W, H, методов классификации)
- Анализа ошибок модели

## Пример

Создайте ноутбук `exploration.ipynb` для базового исследования данных.
